# Career Compass

Career Compass is a comprehensive web application designed to guide users through their career paths by providing personalized roadmaps, skill recommendations, interview questions, and resume generation. Built using the MERN stack, this platform empowers users to enhance their skills and plan their career journeys efficiently.

---

## Tech Stack

* **Frontend:** React.js, Tailwind CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Hosting:** Render, Vercel, or Netlify

---

## Features
(Currently in progress )
* **Personalized Roadmaps:** Users can get career roadmaps based on the role they are interested in.
* **Skill Guidance:** Users can view skills required for specific roles.
* **Course Recommendations:** Suggests courses to enhance user skills based on their interests.
* **Interview Preparation:** Provides commonly asked interview questions for specific roles.
* **Resource Sharing:** Users can submit helpful resources like roadmaps, interview questions, and course recommendations.
* **Admin Panel:** Admin can manage shared resources and monitor user activity.
* **Career Chatbot:** AI-powered chatbot to answer career-related queries.
* **Authentication:** Signup, login, and user profile management.
* **Resume Generation:** Users can generate resumes by entering basic details.
* **Progress Dashboard:** Allows users to track and mark skills as complete.

## Usage

1. **Signup/Login:** Create an account or login to access all features.
2. **Generate Resume:** Enter basic details to generate a professional resume.
3. **Explore Career Paths:** Access roadmaps, skill recommendations, and interview questions (upcoming features).
4. **Share Resources:** Submit helpful courses, roadmaps, or interview questions.
5. **Track Progress:** Mark skills as complete and view your progress dashboard (upcoming feature).
